**AI-Driven Job Transformation by 2030**

An interactive data analytics project that explores how Artificial Intelligence may reshape jobs by the year 2030.
This project analyzes automation risk, AI exposure, salaries, education levels, and workforce trends using Excel Dashboarding,Data Analysis and Machine Learning models.


**Project Overview**

Artificial Intelligence is rapidly transforming industries by automating repetitive tasks, improving efficiency, and changing traditional job roles.
This project aims to identify which jobs are at higher risk of automation and what factors influence that risk.
Using a real-world dataset, this project converts raw data into meaningful insights through dashboards, charts, KPIs, and predictive modeling.

** Objectives**

- Analyze job roles vulnerable to automation by 2030  
- Study salary trends vs experience  
- Evaluate AI Exposure Index across industries  
- Understand education level impact on automation risk  
- Build interactive dashboards for easy filtering  
- Predict automation probability using Machine Learning

**Key Insights**

- Nearly **50% of tasks** may be automatable by 2030  
- Many jobs fall under **Medium Risk**, meaning transformation rather than replacement  
- Higher education lowers risk, but doesn't eliminate it  
- Experience alone does not guarantee job safety  
- Creative and analytical roles remain more resilient
